
# 🎓 Student Management System (Java)

A simple **console-based Student Management System** developed using **Java**.
This application allows users to manage student records with features like adding, viewing, searching, updating, and deleting student details.
All student data is stored using **file handling**.

---

## 🚀 Features

* ➕ Add new student
* 📋 View all students
* 🔍 Search student by ID
* ✏️ Update student details
* ❌ Delete student record
* 💾 Data stored in `students.txt` file

---

## 🛠 Technologies Used

* Java
* Object-Oriented Programming (OOP)
* File Handling
* Eclipse IDE

## ▶ How to Run the Project

1. Clone or download this repository.

2. Open the project in **Eclipse IDE** or any Java IDE.

3. Compile and run the main file:

StudentManagementSystem.java

4. Follow the console menu options to manage students.
